6. 标准差¶

  • S = [s1, s2, ..., sn]
  • m = (s1+s2+...+sn)/n
  • D = [d1, d2, ..., dn], di = si-m
  • Q = [q1, q2, ..., qn], qi = di^2
  • v = (q1+q2+...+qn)/n
  • s = sqrt(v)
  • v' = (q1+q2+...+qn)/(n-1)
    • ${s^2} = \frac{1}{{n - 1}}(\sum\limits_{n - 1}^n {({x_i} - \bar x){)^2}}$
    • numpy.std(S)
  • s' = sqrt(v')
    • numpy.std(S, ddof=1)
    • ${s^2} = \frac{1}{{n - 1}}\sqrt {(\sum\limits_{n - 1}^n {({x_i} - \bar x){)^2}} }$
